## Installation

> nuxt-dropzone for Nuxt3 applications.

## Quick Start

```sh
npm i nuxt-dropzone
```

```sh
yarn add nuxt-dropzone
```

## Setup

### Installation

Then, add six-dropzone to the modules section of nuxt.config.js:

### Config

***nuxt.config.js***

``` js
{
     modules: [
        'nuxt-dropzone',
    ]
},

```
